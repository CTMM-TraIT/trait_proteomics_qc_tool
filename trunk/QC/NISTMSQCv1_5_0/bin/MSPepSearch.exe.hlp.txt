Options: [{sdfm}][j][r][p][x][i][o][q][n][v][{leh[n]}[{IQSHLMPG}] [/Z z] [/M m] [/PATH path] [/LIB lib] [/WRK wrk] /INP InputFile [/OUT[TAB] OutputFile] [/HITS max_hits] [/MinMF minmf]

Search Method Options

 search type:
   I identity                 Q quick identity
   S simple similarity        H hybrid similarity
   L neutral loss similarity  M ms/ms in EI library
   P peptide ms/ms search in a ms/ms library (use peak annotations and weighting)- Default
   G generic ms/ms search in a ms/ms library (no peak annotations or weighting used)

 presearch type:
   s sequential (compare to all library spectra)
   d standard (use pre-indexed peaks) - Default
   f fast presearch (use pre-indexed peaks)
   m precursor ion m/z within precursor ion uncertainty
      - only for MS/MS search (P or G)

 search options:
   r reverse search
   p penalize rare compounds (I in Mainlib & Replib only)
   x do not output hit probabilities
 ms/ms search (P or G) options:
   j hit reJection; Pj: reject non-peptide hits, Gj: reject peptide hits
   a alternative peak matching in computing match factors (experimental)
   v output rev-dot (reverse match factor))
   i ignore peaks within max(z,m) m/z units of precursor m/z, see /Z, /M option
   l e h[n] ms/ms search threshold: l Low, e mEdium, h High(default), hn (n=1..9) Higher
 peptide ms/ms search (P) scoring options:
   o q n : o Use OMSSA score; q Q-tof; n use number of replicates

 search parameters:
   /Z z = precursor ion uncertainty, m/z units (default=1.6, only for P, G)
   /M m = ms peak m/z uncertainty (default=0.6, only for P, G)
   /MzLimits  n1 n2   compare peaks with n1 <= m/z <= n2; default P, G: 1 -1, others: -1 -1
   /MzMinFromSpec     n1 = min. m/z from search spectrum, n2 = -1
   /MzMinFromPrecursorMZ  n1 = PrecursorMZ/3 from search spectrum, n2 = -1
                  If several n1 values have been requested then the min. value is used
   /MinInt    n       = min. peak intensity (1-999), default: 1
   /MwForLoss n       = mw of the unknown, for Loss, Hybrid, or MS/MS in EI lib (L,H,M) only

Input, Output, and Other Options

 /PATH  path = path to folder containing MS libraries
 /MAIN  main = [path]name of NIST Main MS library (one or none)
 /REPL  repl = [path]name of NIST Replicate MS library (one or none)
 /LIB   lib  = [path]name of other ms libraries (may be more than one)
 /WRK   wrk  = full path to work folder; default = current folder
 /INP InputFile            = input .MGF (Mascot generic) or .MSP (NIST MS)-type file
 /OUT[TAB][N|A] OutputFile = output text file [tab-delimited, only for P]; [N]ew (default) [A]ppend; default = stdout
 /OUTMGF[N|A]   OutputFile = output MGF text file containing search spectra
 /HITS max_hits            = max. number of output hits; default = 3; up to 100; for m or s presearch up to 400
 /MinMF minmf              = min. match factor (MF) to output
 /TIME            output to stderr each search time in seconds
 /PROGRESS        output to stderr progress messages
 /PWND hWnd       hexadecimal handle of a window to which will be sent registered
                  messages "WM_MSPEPSEARCH_MSG" with WPARAM=0, LPARAM=%completion
 /NumCompared     output number of compared spectra
 /OnlyFound       do not output empty hit lists (default)
 /OnlyNotFound    output only empty hit lists
 /All             output all hit lists
 /MaxNumSpec  num      = max. number of spectra to input
Output additional columns in OUTTAB output
 /COL c1[,c2][,c3]... where c# are two letter combinations in {} below, no spaces around comma
 /OutPrecursorMZ       {pz} include precursor m/z in the output
 /OutDeltaPrecursorMZ  {dz} include Hit-Unknown precursor m/z difference
 /OutPrecursorType     {tz} include precursor type (e.g. [M+H]+) in OUTTAB output
 /OutChemForm          {cf} include chemical formula (M) in OUTTAB output
 /OutSpecNum           {sn} output input spectrum number
 /OutCAS[rn]           {cn} output CAS registry number in OUTTAB output
 /OutNISTrn            {nn} output NIST registry number in OUTTAB output
 /OutInstrType         {it} output Instrument Type in OUTTAB output
 /OutCE                {ce} output Collision Energy in OUTTAB output
 /OutMaxScore          {xs} output Hit list max Score in each hit in OUTTAB output
 /HiPri                Elevate program execution priority from Normal to Above Normal
 /LibInMem             Load libraries in memory (ms/ms search only)
